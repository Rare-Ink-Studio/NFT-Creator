# Rare Ink Studio NFT Creator

Standalone creator for image, audio, video, and paired audiovisual NFT collections. Minting is handled separately through One Home.

## Public website

Open `index.html` or publish this folder with GitHub Pages, Netlify, or another static host.

## Features

- Simple edition ZIP generation
- Multi-image metadata and ZIP generation
- Advanced layered randomizer
- Multi-artist profiles, filters, combination rules, and credits
- Trait rarity, layer chance, trait rules, and exact 1/1 controls
- Project JSON save, load, and browser autosave
- PNG and metadata ZIP export
- Audio NFT editions with optional cover art
- Video NFT editions with optional poster art
- Paired audiovisual NFT editions

Artwork remains in the visitor's browser and is not uploaded to the host.
